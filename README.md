# FoodieFind

FoodieFind is a full-stack food discovery web app that helps users find real nearby food places, save favorites, write reviews, and manage imported restaurant data.

## Features

- User registration and login
- Real nearby food place search using browser location
- Geoapify/OpenStreetMap nearby restaurant discovery
- Save real nearby places into FoodieFind
- Favorites system
- Restaurant detail pages
- User reviews and ratings
- User profile page
- Admin restaurant management
- Modern dark UI with responsive layout

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

### Backend
- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL
- JWT authentication
- Geoapify API
